# 網站架構

## 概覽

本專案使用 Docker 容器化部署，包含以下服務：
- **Nginx**: 反向代理伺服器
- **Blog**: 部落格 Flask 應用程式
- **Shop**: 商城 Flask 應用程式
- **SQL Server**: 資料庫

```
                    ┌─────────────┐
    使用者 ────────► │    Nginx    │ (port 80)
                    │  反向代理    │
                    └──────┬──────┘
                           │
              ┌────────────┼────────────┐
              │            │            │
              ▼            ▼            ▼
        ┌──────────┐ ┌──────────┐ ┌──────────┐
        │  Blog    │ │  Shop    │ │  DB      │
        │ Flask    │ │ Flask    │ │SQL Server│
        │ port 5000│ │ port 5001│ │ port 1433│
        └──────────┘ └──────────┘ └──────────┘
```

## 目錄結構

```
MyBlogWebSite/
├── docker-compose.yml          # Docker 服務编排
├── nginx/
│   ├── Dockerfile              # Nginx 映像檔
│   └── conf.d/
│       └── nginx.conf          # Nginx 設定檔
├── blog/                       # 部落格應用程式
│   ├── app.py                  # Flask 主程式
│   ├── requirements.txt        # Python 依賴
│   └── ...
├── shop/                       # 商城應用程式
│   ├── app.py                  # Flask 主程式
│   ├── requirements.txt        # Python 依賴
│   └── ...
└── init-scripts/               # SQL Server 初始化腳本
    └── init.sql
```

## Container 規劃

| Container | 用途 | 映射 Port | 環境變數 |
|-----------|------|-----------|----------|
| `nginx` | 反向代理伺服器 | 80, 443 | - |
| `blog` | 部落格 Flask 應用 | 5000:5000 | FLASK_ENV, DB_HOST, DB_PASSWORD |
| `shop` | 商城 Flask 應用 | 5001:5001 | FLASK_ENV, DB_HOST, DB_PASSWORD |
| `db` | SQL Server 資料庫 | 1433:1433 | SA_PASSWORD, ACCEPT_EULA |

## Nginx URL 路由

- `/blog/*` → 轉送到 `blog:5000`
- `/shop/*` → 轉送到 `shop:5001`
- `/` → 預設導向部落格 `/blog/`

## 實作步驟

1. **建立 docker-compose.yml** - 定義 4 個服務
2. **建立 Nginx 設定** - 設定反向代理和路由規則
3. **建立 Blog Flask 應用** - 基本部落格結構
4. **建立 Shop Flask 應用** - 基本商城結構
5. **建立 SQL Server 初始化腳本** - 建立資料庫和表格
6. **設定網路** - 讓 containers 互相溝通

## 資料庫設定 (預設值)

- **主機**: db
- **連接埠**: 1433
- **帳號**: sa
- **密碼**: (環境變數設定)
- **資料庫名稱**: BlogShopDB
